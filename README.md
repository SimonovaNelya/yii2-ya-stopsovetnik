Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist skeeks/yii2-ya-stopsovetnik "*"
```

or add

```
"skeeks/yii2-ya-stopsovetnik": "*"
```
